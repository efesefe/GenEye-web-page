App Features

Brief description of Gen-Eye features.

Object Detection

Providing high speed and high accuracy object detection only using the phone camera. Choose one of the many objects that can be detected and start searching.

Obstacle Detection

GenEye uses your phone's camera to warn you against the dangers of the physical world. If there are any obstacles in the way it will warn you with a voice output.
