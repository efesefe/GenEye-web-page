Our Services

In our daily life, Vision is important for us to comprehend the environment. In addition to this, humans’ vision has 87% ratio of understanding the perception. Unfortunately, visually impaired people have major obstacles in their ways. They encounter different kinds of setbacks too. There are some solutions for their problems but those are usually inefficient or insufficient for visually impaired people.

The reason behind GenEye is to create a better solution for visually impared to use in their smartphones which can be easily accessible and simple to use. Furthermore, GenEye will have Machine Learning algorithms to improve itself and understand the surroundings. GenEye will have a server system easily accessible for image processing systems and data banks.

  • Object Detection
  • Obstacle Detection

Object Detection

GenEye, provides voice interaction service for the users. users can say the name of the object they need to find and the system will be looking for that object.

If found, system will inform the user with a voice notification. With the program’s guidance, users should be able to allign the searched object by center of the screen which will help them to realize the exact direction of the object.

Obstacle Detection

Obstacle Detection service also works as real time processing system. The program will warn the user with voice warning if there is an obstacle on the way.

During a daily walk routine, there may be obstacles that can be harmful to the visually impared. By notifying them of their surroundings, program will improve visually impaired user's lifes
